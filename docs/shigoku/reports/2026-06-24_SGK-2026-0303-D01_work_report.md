# D01: Ollama互換層の完全削除 作業報告書

SGK-2026-0303 のサブタスク D01: Ollama互換層の完全削除。指定された3ファイルの変更を実施。

## 実施内容

### 1. `src/core/llm/local_provider.py` の削除
- ファイル全体を削除（`LocalLLMProvider`, `TaskComplexityClassifier` クラスを含む）

### 2. `src/core/llm/__init__.py` のクリーンアップ
- `LocalLLMProvider`, `TaskComplexityClassifier` の import を削除

### 3. `src/core/gpu_accelerator.py` の Ollama メソッド群削除
以下の deprecated メソッドを削除:
- `_ollama_deprecated_check()` - 内部ヘルパー
- `is_ollama_available()`
- `get_installed_models()`
- `pull_model()`
- `list_ollama_models()`
- `pull_ollama_model()`
- `query_ollama()`

付随する変更:
- `__init__` から `_ollama_available` フィールドを削除
- import から `shutil`, `warnings`, `List` を削除
- モジュール docstring から「ローカルLLM (Ollama)」を削除
- クラス docstring から Ollama モデル名の記述を削除
- `get_status()` から `ollama_available` と `installed_models` キーを削除、代わりに `gpu_enabled` を追加
- `query_llm()` の `model` パラメータ型ヒントを `Optional[str]` に修正
- `query_llm()` の docstring から deprecated メソッドへの参照を削除

保持したメソッド:
- `detect_gpu()`, `is_gpu_available()`, `disable_gpu()`, `enable_gpu()`
- `query_llm()`, `get_embedding_device()`, `get_recommended_workers()`
- `get_status()`, `get_gpu_accelerator()`
- `GPUInfo` dataclass, `RECOMMENDED_MODELS`
