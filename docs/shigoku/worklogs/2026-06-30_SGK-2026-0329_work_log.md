# SGK-2026-0329 作業ログ

## 作業日時: 2026-06-30

### 実施内容
1. `rules/lessons.md`、`rules/codingrules.md`、`rules/python-tests.md`、`rules/task-ledger.md`、`rules/shigoku-docs.md` を確認し、設計方針を確定した。
2. 既存コードベースの調査: `request_guard.py`（HTTP/HITL adapter）、`enhanced_ethics_guard.py`（破壊的 SQL パターン）、`SmartRequest`（ネットワーク送信ラッパー）、`smart_sqli.py` / `smart_xss.py` / `smart_cmd_ssrf.py`（injection specialist）の現状を棚卸した。
3. `src/core/security/execution_safeguard.py` を新規作成:
   - `SafeguardDecision` dataclass（allow/deny/hitl_required ファクトリメソッド）
   - `PayloadRiskPolicy`（DELETE FROM / UPDATE SET / INSERT INTO / DROP TABLE / TRUNCATE / ALTER の正規表現検出 + time-based 検出フック）
   - `MethodRiskPolicy`（GET 安全 / POST-PUT-DELETE-PATCH 要 HITL / CTF 全許可）
   - `ExecutionSafeguardService`（method → payload の順で評価、bugbounty で fail-closed）
   - シングルトンファクトリ `get_execution_safeguard()` / `reset_execution_safeguard()`
4. `request_guard.py` の docstring を更新し、HTTP/HITL adapter としての責務を明示した。
5. `__init__.py` に全公開シンボルを追加した。
6. `smart_request.py` に `execution_safeguard` パラメータを追加し、safeguard 優先パスと reason_code 付きログを実装した。
7. `smart_sqli.py` / `smart_xss.py` / `smart_cmd_ssrf.py` のデフォルト mode を `ctf` → `bugbounty` に変更し、`get_execution_safeguard()` 経由で `SmartRequest` を作成するよう修正した。
8. `tests/core/security/test_execution_safeguard.py` に 52 件の targeted unit test を作成した:
   - SafeguardDecision ファクトリ / デフォルト値
   - PayloadRiskPolicy（bugbounty で destructive SQL 遮断 / CTF で許可 / 大文字小文字 / dict flatten / time-based フック）
   - MethodRiskPolicy（GET 安全 / POST 要 HITL / CTF 全許可 / is_aggressive ヘルパー / デフォルト bugbounty）
   - ExecutionSafeguardService（デフォルト bugbounty / GET 常時許可 / POST HITL 承認/拒否 / キャッシュ / callback 不在 fail-closed / CTF 全許可 / destructive SQL 遮断 / 例外時 fail-closed / reason_code / matched_rules）
   - シングルトンファクトリ（作成/モード切替/callback 更新）
   - SmartRequest 統合（safeguard パス / destructive payload 遮断 / legacy guard fallback）
   - SmartSQLiHunter 統合（デフォルト bugbounty / CTF override）
9. targeted test を実行し、全 52 件が GREEN であることを確認した。
10. 既存回帰テスト（request_guard 5件 / smart_request 3件 / injection 380件）を実行し、全件 PASS を確認した。
