# CLI-First Ops Routing

Use this file for operations involving:

- report
- session
- validate
- gate checks
- SHIGOKU operational commands

## Preferred routing

For `report`, `session`, or `validate` operations, use `shigoku-ops` first.

Preferred forms:

```bash
.venv/bin/shigoku-ops ...
python3 scripts/shigoku_ops_cli.py ...
```
